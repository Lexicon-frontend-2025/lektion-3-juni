Nestad CSS

  • Strukturera din CSS som HTML

BEM

  • Ett sätt att skriva strukturerad och tydlig CSS
  • Varje del av komponenten får ett eget namn
  • Bättre översikt i större projekt
  • Förenklar namngivning

Avancerade selektorer

  • & - referens till dig själv
  • :is() - en funktionell pseudo-klass; flera alternativ, en stil
  • :where() - bra för resets eller generella regler
  • :has() - "Om mitt barn är detta - styleas jag om"

Specificitet

  • Element - 1p
  • Klass - 10p
  • Id: 100p
  • Inline: 1000p

Pseudo-element

  • :hover - när muspekaren är över elementet
  • :active - när du klickar ner musknappen
  • :focus - när ett element är fokuserat (t.ex input)
  • :focus-visible - fokus endast när det är relevant för tillgänglighet
